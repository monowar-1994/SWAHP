Before You Upload
Single Record

  • Click on the "Upload Single Record" On the Menu Bar.
  • Read The description text in each box before uploading the value.
  • If you are updating a date field please use the calendar on the left side of the input box. This will avoid any kind of accidental date formatting issue in the database.
  • If you are unsure about any field, please contact your research supervisor or system administrator.
  • Check after 5 minutes of uploading a record to see if it is in permanent database.

Before You Upload Batched Records

  • Do not use spreadsheets that you created. Use the one provided in the page of "Upload Batch Record"
  • I am currently looking into a work around so that it can handle more than 2GB of zipped files at once. I am looking into Microsoft Azure Cloud.
  • After successful batch upload message shown in your window, please check back 15 minutes later and see if the records are successfully dumped in the permanent database. If it is not there, contact system administrator immediately as it can be an indicator of catastrophic/irrecoverable system failure.
